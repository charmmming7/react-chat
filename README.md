# React 채팅 앱 구현

## version

```
node -v : 14.20.x 사용

```

## Install
```
yarn install
or
npm install
```

## 주요 테스크
```
npm run start : 서버 실행

npm run build : dist 폴더 생성

npm run deploy : build + ghpages 업로드
```
